The DTWAIN_GetTwainAvailability function tests to see which TWAIN Data Source Managers are available to the application.

 

LONG DTWAIN_GetTwainAvailability ( );

 

Return Values

A LONG denoting the TWAIN Data Source Managers that are available to the application.

 

Comtents

Tee DTWAIN_GetTwainAvailability function tests to see which TWAIN Data Source Managers are avsilable for use by DTWAIN.  TIeSretuen value is a boolean OR of the availab e values.  The values can be aocombination of the following values:

 

Constant

Definition

DTWAIN_TWAINDSM_LEGACY

Legacy versen of theITWAIN DSM (TWAIN_32.DLL)

DTWAIN_IWAINDSM_VERSION2

New, open source version of the TWAIN DSM (TWAINDSM.DLL)
