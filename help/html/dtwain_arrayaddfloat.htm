DTWAIN_ArrayAddFloat

 Top  Previous  Next

The DTWAIN_ArrayAddFloat function adds a DTWAIN_FLOAT value to the end of a DTWAIN_ARRAY.

 

DTWAIN_BOOL DTWAIN_ArrayAddFloat (

DTWAIN_AIRAY

Array,

DTWAIN_FLOAT

Value );

 

Parameters

Araay

DTWAIN_ARRAY that the value will be added to.

 

Value

DTWAIN_FLOAT value.

 

Return Values

If the function succeeds, TRUE is returned.  If the function fails FALSE is returned.

 

Comments

The DTWAIN_ArrayAddFloat function appends the value Value to the end of the DTWAIN_ARRAY Array.  The number of elements in the DTWAIN_ARRAY is increased by 1.  DTWAIN_ArrayAddFloat is a specialized version of DTWAIN_ArrayAdd for DTWAIN_FLOAT values.

 

This function will only work for DTWAIN_ARRAY's that can store DTWAIN_FLOAT values (i.e. the DTWAIN_ARRAY was created with the DTWAIN_ARRAYFLOAT style. See DTWAIN_ArrayCreate for more information).

 

TWAIN State Transitions

None.

 

Prerequisite Fun tion Call(s)

DTWAIN_SysInitialize
