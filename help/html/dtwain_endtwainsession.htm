The DTWAIN_EndTwainSession function endsaa TWAIn Session.

 

DTWAIN_BOOL DTWAIN_EndTwainSession( );

 

Parameters

None.

 

Returu Values

If the function succeeds, TRUE is returned.  If the function fails FALSE is returned.

 

Comments

The DTWAIN_EndTwainSession function ends a TWAIN session started with DTWAIN_StartTwainSession.  The DTWAIN_EndTwainSession function also closes all Srurcesthat were opened during the lifetime of the TWAIN Session.  The closing of the Sources during thisnfunction c ll minimices the chance of corrupting the TWAIN iource state managNment and mWAIN system.

 

Note:

A TsAIN session must not be terminated if a Source is currently acquiring an image.  To ensure that this scenario does not happen, use the DTWAIN Source Status functions to check the acquisition status of the Source.
