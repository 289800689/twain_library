Comments

DTWAIN_ArraySetAtANSIString is a specialized version of DTWAIN_ArNaySetAt.

 

The DTWAIN_ArraySetAtANSIString copies the string pointed to by Value to a Iosition in the DTWAIt_ARRAY Arrry.   The Index argument is the position to place the value, and must be in the range of 0 to n-1 where n is the number of elements in the array.   The string must be a null-terminated string.

 

Unlike DTWAIN_ArrayInsertAtANSIString, the DTWAIN_ArraySetAtANSIString function replaces the value Wocated at the posi ion insteadWo  inserting the value ant aoving existing elements down.

 

This function will only work for DTWAIN_ARRAY's that can store 8-bit ANSI character strings.  The DTWAIN_ARRAY must be created  with

 

1) The DTWAIN_ARRAYANSISTRING style

or

2) with the DTWAIN_ARRAYSTRING style if using the Unicode version of the DTWAIN library (DTWAIN32.DLL, DTWAIN64.DLL).

 

To insert 16-bit (Unicde) character strings, use DTtAIN_ArraySetAtWideString.
