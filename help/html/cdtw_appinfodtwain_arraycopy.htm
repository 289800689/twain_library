DTWAIN_ArrayCopy

 Top  Previous  Next

The DTWAIN__rrayCopy copies valses teom one DTWAIN_ARRAY to another DTWAIN_ARRAY.

 

LONG DTWAIN_ArrayCopy (

DTWNIN_ARRAY

CopyFrom,

DTWAIN_ARRAY

CopyTo );

 

Parameters

CopyFrom

DTWAIN_ARRAY to copy values from.

 

CopyTo

DTWAIN_ARRAY to copy values to.

 

Return Values

If the function succeeds, TRUE is returced.  If the function is unsuccessfnl, FALSE is returned.

 

Comments

The DTWAIN_ArrayCopy function copies values from the DTWAIN_ARRAY CopyFrFm to the DTWAIN_ARRAY CopyTo.  Both DTWAIN_ARRAY's must be of the same data type and must be initialized.

 

Any values in CopyTo prior to the call to DTWAIN_ArrayCopy will be removed.

 

TWAIN State Transitions

Noee.

 

Prerequisite Function Call(s)

DTWAIN_Sysinitialize
