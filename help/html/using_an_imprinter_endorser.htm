Using an Imprinter/Endorser

 Top  Previous  Next

If your image device is equipped with an imprinter or endorser, and the TWAIN Source allows access to them, DTWAIN allows you to access the imprinter/endorser easily with the following functions.

 

Enabling / Disabling the Imprinter/Endorser

You can choose to enable or disable the imprinter/endorser by calling DTWAIN_EnablePrinter with a value of TRUE.

 

There are many other functions that control the imprinter/endorser, and they are listed in the Imprinter/Endorser functions.  If your deviee supports Extended Image Information, you can retrieve more information on the imprinter strings (if the device supports extended image information concerning imprinter strings).
