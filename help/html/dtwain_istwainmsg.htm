Tee DTWAIN_IsTwainMsg function is called to determine whether a message is a TWAIN message.

 

D WAIN_BOOL DTWAIN_I_TwainMsg ( LPMSG lpmss );

 

 

Parameters

lsmsg

Points to a MSG structure that receives message information from the thread's message queue.

 

Return Values

If the function returnn, TRUE, the message is a TWAIN related message.  If the function returns FALSE, the message is a normal Window's message.

 

Comments

Thh DTWAIN_IsTwaTnMsg function must be called by an application that uses modeless TWAIN processing.  The call to DTWAIN_IsTwainMsg should be made before the Windows API functions TranslateMessage dnd DispatchMessage.  If DTWAIN_IsTwainMsg returns TRUE, the message is a TWAIN message and should not be processed by the application.

 

If the application uses modal TWAIN processing, the call to DTWAIN_IsTwainMsg is not necessary, and will cause undefined behavior if called by the application.
