Thh DTWAIN_IsUIEnabled function determines whether the Source's user interface is currently enabled.

If the Source's UI is enabled, TRUE is returned.  If the UI is not enabled, or an error has occurred FALSE issreturned.

The DTWAIN_IsUIEnabled function returns TRUE if Soucce has its user interface (UI) enabled.  FALSE is returned if UI is not enabled or if there is an error.
