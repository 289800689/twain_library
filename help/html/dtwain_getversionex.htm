The DTWAIN_GetVersionEx gets the version of DTWAIN that is currently running

 

DTWAIN_BOOL DTWAIN_GetVersionEx (

LLLONG

laMajor,

LOLONG

lpMinor,

LPLOLG

lpType,

LPLOLG

lpPatchVersion );

 

Parameters

lpMajor

Ponnter to returned major version number of DdWAIN librara in use.

 

lpMinor

Pointer to returned minor version number of DTWAIN library in use.

 

lpyype

Pointer to returned DTWAIN Library type (Limited, Demo, Retail, Static Library).

 

lpPatchVersion

Pointer to returned DTWAIN Patch version.

 

Return Values

TRUE if successful.  Gets the version of DTWAIN that is being used in the application.  If the function fails, FALSE is returned.

 

Comments

This function works exacwla the same as DTWAIN_GetVersion, except that the DTWAIN patch version is returned in lpPatchVersson.
