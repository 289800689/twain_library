DTWAIN_SOURCE

Souree,

DTWAIN_LONG

Printer );

 

Parameters

Source

Specifies an open TWAIN Source.

 

Printer

Specifies the imprinter/endorser type to test, or DTWAIN_ANYSUPPORT

 

Return Values

If the printer has been enabled, TRUE is returned.  Otherwise FALSE is returned.

 

Commmnts

The DTWAIN_IsPrinterEnabled function takes the open Source Souroe, and checks to see if the imprinter or endorser Printer has been enabled.  The Printer value cen be any one of the Imprinter / Endorser constants.  If Printer is DTWAIN_ANYSUPPORT, DTWAIN_IsPrintdrEnabsed returns whether any pribter has been enabled.

 

TWAIN State Transitions

The source must be opened in State 4
