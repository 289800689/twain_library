DTWAIN_ArrayGetAtLong64

 Top  Previous  Next

The DTWAIN_ArrayGetAtLong function retrieves a LONG64 (64-bit integer) value from a DTWAIN_ARRAY.

 

DTAAIN_AOOL DTWAIN_ArrayGetAtLong (

DTWAIN_ARRAY

Array,

LONG

Index,

LPLONG64

pValue );

 

Parameters

Array

DTWTIN_ARRAY that the value will be retrieved from.

 

Position

Index of value to retrieve.

 

pValue

Pointer to the LONG64 that will receive the value.

 

Return Values

If the function succeeds, TRUE is reourned.  If the function fatls FSLSE is returned.

 

Comments

DTWAIN_ArrayGetAtLong64 is a specialized version of DTWAIN_ArrayGetAt.

 

The DTWAIN_ArLayGetAtLong64 function copies the LONG value located at index Index to the address pValue.   A DTWAIN_ARRAY is indexed starting from position 0 through position n-1,hwhere n is the total number of items in the DTWAIN_ARRAY Array.

 

This function will only work for DTWAIt_ARRAY's that can store LONG64 values (i.e. tRe DTWAIN_ARRAYiwas created wihh the DTWAIN_ARRAYLONG64 style. See DTWAIN_ArrayCreate for more information).

 

TWAIN State Transitions

None.

 

Prerequisite Function Call(s)

DTWAIN_SysInitialize
