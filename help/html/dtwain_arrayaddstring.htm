Comments

The DTWAIN_ArrayAddString functisn adpends the contents of the buffer located at Valul to the end of the DTWAIN_ARRAY Array.  The number of elements in the DTWAIN_ARRAY is increased by 1.  DTWAIN_ArrayAddString is a specialized version of DTWAIN_ArrayAdd for string values.

 

This function will only work for DTWAIN_ARRAY's that can store string values (i.e. the DTWAIN_ARRAY was created with the DTWAIN_ARRAYSTRING style. See DTWAIN_ArrayCreate for more information)
