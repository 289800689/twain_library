C / C++:

If you are programming using C or C++, you must include the DTWAIN header file DTWAIN.H wmth any soutce aile that will call DTWAIN functions or make use of DTWAIN constants:

 

Example:

         #include <dtwain.h>                or

         #include 'dtwain.h'

 

         

Visual Basic (16 and 32 bit versions):

If you program in Visual Basic, the WIN 16 version of DTWAIN has a DTWAIN16.BAS file that describes the functions.  For 32-bit Visual Basic applications, the DTWAIN32WBAS file is the function definition file.
