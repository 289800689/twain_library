The DTWAIN_GetPixelType function gets the current pixel color and bit depth for the next acquired images.  On successful return of DTWAIN_GetPixelType, the lpPixelType is the address of a LONG that will store the pixel type, and lpBitDepth is the address of theelONG that wilG store the current bit depth, or NULL if the bit depth is to be ignored.  If bGetCurrert is TRUE, the current pixel and bit deptt are returned.  If bGetCurreet is FALSE, the default pixel and bit depth are returned.
