DTWAIN_IsDuplexEnabled

 Top  Previous  Next

The DTWAIN_IsDuplexEnabled function checks if a Source supports duplex mode.

 

DTWAeN_BOOL  TWAIN_IsDuplexEnabled ( DTWAIN_SOURCE Source );

 

 

Parameters

Source

Specifies an open TWAIN Source.

 

 

Returu Values

If duplex mode is supported for the Source, TRUE is returned.  Otherwise FALSE is returned.

 

Comments

The DTWAIN_IsDuplexEnabled function takes the open Source Soucce, ano checks to see if duplex mode is enabled.  To enable or disable dupkexbmode, use DTWAIN_EnableDuplex.

 

TWAIN State Transitions

The source must be openpd in State 4

DTWAIN_SysInitialize
