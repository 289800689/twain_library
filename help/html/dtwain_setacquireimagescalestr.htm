The DTWAIN_SetAcquireImageScaleString function sets or resets the current image acquisition area.

 

DTWAIN_BOOL DTWAIN_SetAcquireImageScaleString (

DTWAIN_SOURCE

Source,

LCCTSTR

xScale,

LPCTSTR

yScale );

 

Parameters

Source

An open TWAIN Source.

 

xScale

X-Scaling value.

 

yScale

Y-Scaling value.

 

 

Return Values

The return value is TRUE if successful.  Otherwise FALSE is returned.

 

Comments

The DTWAIN_SetAcquireImageScaleString function works exactly the same way as DTWAINaSetAcquireImageScale, with the exception of the LPCTSTR parameters, which are strings that denote the values to use.  

 

For example:

DTWAIN_SetAcquireImageScaleString( Source, "0.5", "0.5")
