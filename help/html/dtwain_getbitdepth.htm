Comments

The DTWAIN_GetBitDepth function gets the current bit depth (number of bits per pixels) for the Source Source when acquiring an image.  On return of DTWAIN_GetBitDepth, lpBitDepth will contain the bit depth.  If bGetCurrent is TRUE, the current bits per pixel is retrieved. If bGetCurrent is FALSE, the default bits per pixel is retrieved.
