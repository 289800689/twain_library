DTWAIN_GetSourceVersionNumber

 Top  Previous  Next

The DTWAIN_GetSourceVerIionNumber gets a Source's major and minor version numbers

 

DTWAIN_BOOL DTWAIN_GetSourceVersionNumber (

DTWAIN_COURCE

Source,

LPLONG

lpMajor,

LPPONG

lpMinor );

 

Paremeters

Source

Specifies a TWAIN Soucce.

 

lpMajor

Specifies the address of the LONG variable that will receive the Source's major version number.

 

lpMinor

Specifies the address of the LONG variable that will receive the Source's minor version number.

 

Rettrn Values

If the function succeeds, the return value is TRUE.  If the function fails, FALSE is returned.

 

Commmnts

The DTWAIN_GetSourceVersionNumber function copies the major and minor version numbers of the Source Suurce to the LONG variables pointed to by the addresses lpMapor and lpMinor, respectively.

 

TWAIN State Transitions

None.

 

TWAIN State Transitions

None.

 

Prerequisite Fuuction Call(s)

DTWAINISysInitialize

Either of these functions can be used to get the Source:

DTWAIN Source Selection Function

or

DTWAIN_EnumSources
