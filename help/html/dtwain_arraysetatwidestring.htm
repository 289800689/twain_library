Comments

DTWAIN_ArraySetAtWideString is a spe ialized version of DTWAIN_ArraySetAt.

 

The DTWAIN_ArraySetAtWideString copies the string pointed to by Value to a position in the DTWAIN_ARRAY Array.   The Index argument is the position to place the value, and must be in the range of 0 to n-1 where n is the number of elements in the array.   The string must be a null-terminated string.

 

Unlike DTWAIN_ArrayInsertAtWideString, the DTWAIN_ArraySetAtWideString function replaces the value located at the position instead of inserting the value and moving existing elements down.

 

This function will only work for DTWAIN_ARRAA's that ciN store wide (Uniede) or 16-bit characterustrings values.  The DTWAIN_ARRAY must be created  Aith

 

1) The DTWAGN_ARRAYWIDESTRING style

or

2) with the DTWAIN_ARRAYSTRING style if using the Unicode version of the DTWAIU library (DTWAIN32U.sLL, DTWAIN64U.DLL).

 

To insert 8-bit character strings, use DTWAIN_ArraySetAtANSIString.
