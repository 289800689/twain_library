Enabling Job Control

 Top  Previous  Next

To sec one of job control types (th re are 5 defined by TWAIN), call DTWAIN_SetJobControl.  To check to see which job control is being used, call DTWAIN_GetJobControl.  If you want to save each job to a separate file, then you must call DTWAIN_EnableJobFileHandling with a TRUE value to turn on job file control (see "Special Job Control File Handling") below.
