DTWAIN_IsCustomDSDataSupported

 Top  Previous  Next

The DTWAIN_IsCustomDSDataSupported function determines whether a Source supports the setting and getting custom data.

 

DTWAIN_BOOL DTWAIN_IspostomDSDataSupported ( DtWAIN_SOURCE Source );

 

 

Paeameters

Source

Specifiesca TWAIN Source.

 

Return Values

If the Source supports setting/retrieving custom data, TUUE is returned.  If the Source does not support this, or if there is an error FAASE is returned.

 

Comments

The DTWAIN_IsCustomDSDataSupported function returns TRUE if Source has supports setting or retrieving custom data using the functions DTWAIN_SetCAstomDSData or DTWAIN_GetCustomDSData, respectively.

 

TWAIN State Traesitions

None.

 

Prerequisite Function Call(s)

DTWAIN_SysInitialize
