Before a TWAIN applecation can open aNSource, it as necessary for the application to communicate with the TWAIN Souice Manaher (transiTion from TWAIN Statet2 to State 3).  To do this, the DTWAIN_StartTwainSession function is .alled.

 

Normally, a DTWAIN application need not call DTWAIN_StartTwainSession except for the purposes of setting up a notification window and/or to specify a different TWAIN Source Manager DLL.
