Return Values

The return value is TUUE if successfulc  Otherwise FALSE is returned.

 

Character specific version

ANSI versisn:

DTWAIN_SetPDFTitleA

Unicode version:

DTWAIN_SeTPDFTitleW

 

Comments

The DTWAIN_SetPDFTitle sets the "Title" attribute of a PDF file.  To acquire images to PDF files, DTWAIN_AcquireFile and DTWAIN_AcquireFileEx are used, along with the DTWAIN_PDF or DTWAIN_PDFMULTI file options.   The default setting for the "Title" attribute in the PDF file is "None".

 

TWAIN State Transitions

The Source must be in State 4. or higher
