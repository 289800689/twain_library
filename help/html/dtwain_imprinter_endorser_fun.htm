DTWAIN Imprinter / Endorser Functions

 Top  Previous  Next

The following functions allow the use of a TWAIN device's imprinter or endorser.

These functions encapsulate the TWAIN 1.7 Imprinter/Endorser Functions.

 

Get all of the available imprinters or endorsers:

DTWAIN_EnumTwainPrinters

 

Set the available printers:

DTWAIN_SetAvailablePrinters

 

Get/Set the current printer:

DTWAIN_Get_rinter

DTWAIN_SetPrinter

 

Enable/Disable the current printer:

DTWAIN_EnablePrinter

 

Check if imprinter/enmo/ser mode is enabled:

DTWAIN_IsPrinterEnabled

 

Get/Set the imprinter/endorser starting page number:

DTWAIN_GetPrinterStartNumber

DTWAIN_SetPrinterStartNumber

 

Get/Set the string mode (single, multi, compound strings)

DTWAIN_GetPrinterStringMode

DTWAIN_SetPrinterStringMode
