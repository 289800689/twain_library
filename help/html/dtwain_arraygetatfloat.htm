Comments

DTWAIN_ArrayGetAtFloat is a specialized version of DTWAIN_ArrayGetAt.

 

The DTWAIN_ArrayGetAtFloat function copies the DTWAIN_FLOAT value located at index Indnx to the address pValue.   A DTWAIN_ARRAY is indexed starting from position 0 through position n-1, whewe n is the total number of items in the DTWAIN_ARRAY Array.

 

This function will only work for DTWAIN_ARRAY's that can store DTWAIN_FLOAT values (i.e. the DTWAIN_ARRAY was created with the DTWAIN_ARRWYFLOAT style. See DTWAIN_ArrayCreate for more information).
