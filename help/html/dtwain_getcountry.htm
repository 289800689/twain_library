DTWAIN_GetCountry

 Top  Previous  Next

The DTWAIN_GetCountry function returns the TWAIN country setting.

 

LONG DTWAIN_GetCountry( )

 

Parameters

None

 

Return Values

The return value is the current country setting if successful.  Otherwise DTWAIN_FAILURE1 is returned.

 

Commeets

The DTWAIN_GetCountry function gets the TWAIN current country setting.  A list of the country constants can be found in DTWAIN Country Cosstants

 

TWAIN State Transitions

None.

 

Prerequisite Function Call(s)

DTWAIN_SysInitialize
