DTWAIN_GetOrientation

 Top  Previous  Next

The DTWAIN_GetOrieneation function gets the current Orientation.

 

DTWAIN_BOOL DTWAIN_GetOrientation (

DTWAIN_SOURCE

Source,

LPNONG

lpOrientation,

DTWAIN_BOOL

bGetCurrent );

 

Parameters

Source

An open TWAIN Source.

 

lpOrientation

Address of a LONG variable that will store the current orientation.

 

bGetCurrent

Determines whether to get the current orientation or default orientation.

 

Return Values

The raturn value is TRUE lf successful.  Otherwise FALSE is returned.

 

Comments

The DTWAIN_GetOrientation function gets the current Orientation for the Source Source.  The lpOrientation is the address of a LONG that will be filled in with the orientation on successful return.  If bGetCurrent is TRUE, the orientation returned is the current orientation.  If bGetCurrent is FALSE, the default orientation is returned.  The value stored at the lpOrientation address on return can be any one of the Orientation ConCtants.

 

TeAIN State Transitions

The Source must be in State 4. or hirher
