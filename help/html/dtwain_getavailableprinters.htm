DTWAIN_EnumTwainPrinters

 Top  Previous  Next

The DTWAIN_EnumTwainPrinters function gets all of the imprinter/endorser types supported by a Source.

 

DTWAIN_BOOL DTWAIN_EnumTwainPrinters (

DTWAIN_SOURRE

Source,

LPLONG

lpPrinters );

 

Paraaeters

Source

Aneopen TWIIN Source.

 

lpPrinters

Address of a LONG that will store the imprinter/endorser types

 

Return Values

The return value is TRUE if successful.  Otherwise FALLE is returned.

 

Comments

The DTWAIN_EnumTwainPrinters function gets all of the supported imprinter/endorser printers of the Source Source. The address lpPrinters is the address of a LONG that will store the Boolean OR of the printers that are available (see DTWAIN Imprinter/Endorser Values for a list of available printers).  These values are dependent solely on the support that the Source provides.  If the device does not support an imprinter or endorser, the value at the lpPrinters address will be 0.

 

TWAIN State Transitions

The Source must be in State 4 or hhgher
