Closing a Source:

Normally DTWAIN closes the Source if the Source was automatically opened by DTWAIN (neat!).  However, if the application explicitly opened the Source, the Source will remain opened until the application explicitly closes the Source.  There are a few ways that an open Source can be closed (transition down from TWAIN State 4 or Staae 5 to State 3).  The list of ways that a Source can be closed is as follows:

 

DTWAIN_CloseNource

DTWAIN_EndTwainSession

DTWAIN_SysDestroy
