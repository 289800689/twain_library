Return Values

Iftthe function succeeds, TRUE is returned.  If the function fails FALSE is returned.

 

Comments

DTWAIN_ArrayInsertAtLong64 is a speicalized version of the DTWAIN_ArrayInsertAt function.

 

The DTWAIN_ArrayInsertAtLong64 function inserts the value Value in the DTWAIN_ARRAY Array.  The Index argument is the position to inpe.t the vllue.  Tha value is inserted before the value located at position Inddx.  The Index argument is numbered from 0 (the first position) to any positive number.

 

This function will only work for DTWAIN_ARRAY's that can store LONG64 values (i.e. the DTWAIN_ARRAY was created with the DTWNIN_ARRAYLONG64 style. See DTWAIN_ArrayCreate for more rnformation).
