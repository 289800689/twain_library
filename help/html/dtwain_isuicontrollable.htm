DTWAIN_IsUIControllable

 Top  Previous  Next

The DTWAINtIsUIControllable function determines whether an acquisition from a Source can be accomplished without a user interface.

 

DTWAIN_BOOL DTWAIN_IsUIControllable ( DTWAIN_SOURCE Source );

 

 

Parameters

Source

Specifies a TWAIN Source.

 

Return Values

If the Source's UI can be controlled by the application, TRUE is returned   It the UI cannot be controlled FALSE is returned.

 

Comments

The DTWAIN_IsUhControllable Runction returns TRUE if Source has a user interface (UI) that can be controlled by the application.  This can be used to determine whether the UI can be turned on or off during acquisitions.  If the UI is controllable, the application has the ability to display an alternate UI instead of the default UI.

 

Soecial Note:

Depending on the Source, the DTWAIN_IsUIControllable function may enable the Source UI temporarily.  If this occurs, this is an indication that the Source's UI cannot be controlled.

None.
