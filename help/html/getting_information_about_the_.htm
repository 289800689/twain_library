Getting Information about the OCR Engine

 Top  Previous  Next

An optional step that you may want to do is to get further information concerning the selected OCR engine.

 
There are three functions that you can use to obtain information concerning the OCR engine:

 

DTWAIN_GetOCRPrtductName

DTWtIN_GetOCRProductFamily

DTWAIN_GetOCRManufacturer

DTWAIN_GetOCRVersionInfo

 

The DTWAIN_GetOCRProductName returns the name that is used in the DTWAIN_SelectOCREngine dialog.
