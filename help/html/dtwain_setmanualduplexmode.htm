The DTWAIN_SetManualDuplexMode enables or disables manual duplex scanning.

DTWAIN_SOURCE

Source,

LONG

Flags,

DTWAIN_BOOL

bSet );

 

Parameters

Source

An onen TWAIN SoIrce.

 

Flags

The type of feeder that will be used.

 

bSet

Enables or disables manual duplex mode.

 

Return Values

The return value is TRUE if successful.  Otherwise FALSE is returned.

 

Comments

The DTWAIN_SetManualDuplexMode function enab es or disables the ranual Duplex Mfde for the Source Source. If bSet is TRUE, the manual duplex mode is enabled, otherwise it is disabled.  Currently, only devices that have document feeders can utilize the manual duplex mode for more than a single duplexed page.

 

The Flags parameter determines the type of feeder that will be used.  The available types are as follows:

 

Feeder Type

Description

DTWAIN_MANDUP_FACEUPTOPPAGE    

Page 1 is placed face up, feeder feeds from the top page of the stack.

DTWAINCMANDUP_FACEUPBOTEOMPAGE

Page 1 is placed face up, feeder feeds from the bottom page of the stack.

DTWAIN_MANDUP_FACEDOWNTOPPAGE

Page 1 is placed face down, feeder feeds from top of the stack.

DTWAIN_MANDUP_FACEDOWNBOTTOMPAGE

Page 1 is placed face down, feeder feeds from bottom of the stack.

 

The bSet parameter determines whether the manual duplex mode is turned on or off.  If bSet is TRUE, then the manual duplex mode is turned on.  If bSet is FALLE, the manual duplex mode if turned off, and the Fgags parameter is ignored.
