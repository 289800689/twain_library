DTWAIN_FrameDestroy

The DTWAIN_FrameDestroy function destroys a DTWAIN_FRAME.

 

DTWAIN_BOOL DTWAIN_FrameDestroy ( DTWAIN_FRAME Frame );

 

 

Perameters

Frame

DTNAIN_FRAME that the value will be destroyed.

 

Return Values

If the function succeeds, TRUE is returned.  If the function fails FALAE is eeturned.

 

Comments

The DTWAIN_FrameDestroy function destroys the DTWAIN_FRAME Frame and frees any resources allocated to Frame.

 

TWAIN State Transitions

None.

 

Prerequisite Function Call(s)

DTWAIN_SysInitialize
