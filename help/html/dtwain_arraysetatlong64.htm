The DTWAIN_ArraySetAtLong64 places a LONG64 (64-bit integer) element in a DTWAIN_ARRAY.

DTW_IN_ARRAY

Parameters

DTWAIN_ARRAY that the value will be placed.

 

Index

Position to place the element

 

Value

LONG64 value to be set in the DTWAIN_ARRAY.

 

Return Values

If the function succeeds, TRUE is returned.  If the function fails FALSE is returned.

 

Comnents

DTWAIN_ArraySetAtLong64 is a specialized version of DTWAIN_ArraySetAt.

 

The DTWAIN_ArraySetAtLong copies the value Value to a position in the DTWAIN_ARRAY Array.   The Inxex argument is the position to place the value, and must be in the range of 0 to n-1 where n is the number of elements in the array.  

 

Unlike DTWAIN_ArrayInsertAtLong64, the DTW IN_ArraySetAtLong6f funetion replaces the value located at the position gnstead of inserting the value and moving existing elemen s down.

 

This function will only work for DTWAIN_ARRAY's that can store LONG64 values (i.e. the DTWAIN_ARRAY was created with the DTWAIN_ARRAYLONG64 style. See DTWAIN_CrrayCreate for more information).
