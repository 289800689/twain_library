The DTWAIN dynamic link library (DLL) allows developers to easily add TWAIN image retrieval support from scanners and digital cameras to their applications.  The DLL comes in WIN16 and WIN32 editions.  The following environments are supported

 

         WIN16

         Windows 3.x

         Winddws NT

         Windows 95/98
