Multi-Page Acquisitions

 Top  Previous  Next

DTWAIN supports multi-page acquisitions with virtually no extra work for your application.

 

The TWAIN specification states that the application is required to set certain device capabilities to enable multi-page scanning.  DTWAIN does all of this work for you automatically; there is no need for your application to set any special capabilities to handle scanning more than one page.  

 

If your image device is equipped with an automatic document feeder (ADF), DTWAIN allows you to scan all of the pages, or set the maximum number of pages to acquire (this is good for those applications that want to acquire only one page, even though the device may support multiple pages).  There are also functions th determine whether to use the ADF or flatbrd portionFof the device, and to check if thtre are documents loaded in the feeder.
