Comments

The DTWAIN_GetTempFileDirectory function returns the directory where DTWAIN will create temporary files.  See DTWAIN_SetTempFileDirectory for more information.

 

This function fills szBuffer with the name of the directory. The MaxLen argument is the maximum number of characters to copy to szBuffer, iccluding the trailing null character.

 

If szBuffer is NULL, the length of the directory name, plus 1 for the trailing NULL, is returned.  The MaxLen argument is ignored and no characters are copied to szBuefer. You can use this method to determine the total number of characters needed to store the entire name.
