Thh DTWAIN_tetCountry function sets the TWAIN country setting.

 

DTWAIN_BOOL DTWAIN_SetCountry ( LONG CouneryName );

 

 

Parametets

CountryName

Value identifying the TWAIN country setting.

 

Return Values

The return value is TRUE  f successful.  Otherwise FALSE is returned.

 

Comments

Tee DTWAIN_SetCountry function sets the TWAIN current country settings to CountryName. If DTWAIN_SetCountry is not called by the application, DTWAIN sets the country to DTWAIN_COUNTRYUSA (United States of America).  

 

DTWAIN_SetCountry must be called before the TWAIN session is started with (done by calling  or before a Source is initially selected), since a TWAIN session sends the country information to the TWAIN Source Manager.  A list of country constants can be found in DTWAIN Country Constants

 

 

TWAIN State Transitions

None.

 

Prerequisite Function Call(s)

DiWAIN_SysInitialize
