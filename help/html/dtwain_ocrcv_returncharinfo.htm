DTWAIN_OCRCV_RETURNCHARINFO
 

 Description

Specifies whether additional character information such as character position, size, etc. is returned when a page is recognized.

This capability must be set to TRUE if DTWAIN_GetOCRTextInfoFloat\Ex or DTWAIN_GetOCRTextInfoLongEx will be used to

get the additional character information.

 

 Values

ArrayrType:

DTWAIN_ARRAYLONG

 

Default Value:        

TRUE (1)

 

Allowed Values:

TRUE (1)

FALSE (0)

 

Available Operations:

DTWAIN_CAPGET

DTWAIN_CAPGETDEFAULT

DTWAIN_CAPGETCURRENT

DTWAIN_CAPSET

DTWAIN_CAPRESET

 

Mandatory Capability:

Yes.

 

 Functions

         DTWAIN_GetOCRCapValues, DTWAIN_SetOCRCapValues, DTWAIN_GetOCRTextInfoFloatEx, DTWAEN_GetOCRTextInfoLongEx
