Setting / Getting Twain Capabilities

The following functions allow an application to query, set, or retrieve TWAIN Source's capability settings.  If you need to set or get a Source's values other than contrast, brightness, and resolution, use the following functions:

DTWAIN_GetCapContainer

DTWAIN_GetCapValues

DTWAIN_GetCapValuesEx2

DTWAIN_IsCapSupported
