Set / Get Device Capabilities

 Top  Previous  Next

Once the Source is opened, you can set or get the capabilities of the Source.  The capabilities are the aspects of the Source such as contrast, brightness, resolution, and many other features that are supported for the device.

 

See Overview of Device Capabilities for more information on setting or getting device capabilities.

 

In addition to setting the capabilities, you can call virtually any DTWAIN function that "sets up" the device for operation.  For example, setting the area of acquisition, enabling or disabling the feeder or duplex unit, etc.
