DTWAIN Constants

 Top  Previous  Next

The following sections define the various DTWAIN numerical and data type constants that are used in the DTWAIN library.

 

Except for the array data type constants, all of these constants are LONG integer values.  Please refer to the DTWAINC.H file for the actual numerical values of these constants (some tables do include the value, but always refer to the DTWAINC.H file for the values).
